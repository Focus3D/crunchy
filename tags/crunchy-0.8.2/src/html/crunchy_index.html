Welcome to Crunchy!

Use the menu above to navigate. "Home" always brings you back to this page; "Quit Crunchy" closes the browser tab (or window) in addition to ending the program as such.

Suggestions

If we may suggest, you may want proceed with one of the following three possibilities:

  • If you are new to Crunchy, look under the menu "Tutorials" for "Using Crunchy".
  • If you are familiar with Crunchy, and want to have a look at some included Python tutorials, or load some other crunchy-ready tutorials, look under the menu "Tutorials" for "Python tutorials".
  • Finally, if you are very familiar with Crunchy and want to create your own interactive tutorial, look under the menu "Tutorials", "Making Crunchy Tutorials" for the "How to" document.

Trivia

You can change the page style using Firefox (menu: View --> Page Style) ; this will work for the current page only. If you prefer, you can use the options submenu (under "Help") to set a preferred page style.

If you are viewing this page without using Crunchy, then the menu above (shown as a list of links) will not be working. You can use this link to go through the "Using Crunchy" tutorial; unfortunately, this will give you a very poor impression of Crunchy's true capabilities.
