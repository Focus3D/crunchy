Local tutorial: importtests

When Crunchy loads a local tutorial (using the file browser), it adds the path where that tutorial is loaded to sys.path; as a result, any Python module found in the same directory can be imported. In addition, it is possible to specify one or more relative paths, using the meta tag, to be also added to the Python path (see the source of this file for an example). 

>>> # The following is a module found in the same directory as this file.
>>> import hello
>>> hello.hello()
>>> 
>>> # The following is a module found in a relative directory
>>> import bye
>>> bye.bye()
>>> 
>>> # You can view the updated path
>>> import sys
>>> sys.path
