Graphics: plot summary

The following table is a very incomplete summary of available graphics (plot) commands.

Available plot commands
color(col) or
colour(col) or
set_line_color(col) or
set_line_colour(col) 		Sets the default colour using a valid value given as a string. The longer names (set_line_colo[u]r) are included for users who may already be used to that syntax from the graphics canvas. The various options for specifying a color (col) are given in the table below.
prepare_graph(x_range,
y_range, x_axis, y_axis) 		Sets up the various scales and position of axis. The default parameters are:
x_range=(0, 400)
y_range=(0, 400)
x_axis='middle' (other allowed options are 'bottom' and 'top')
y_axis='left' (other option: 'middle').
x_range(x_min, x_max) Sets the horizontal scale for the plot. May not be needed if the appropriate values have been passed to prepare_graph().
y_range(y_min, y_max) Sets the vertical scale for the plot. May not be needed if the appropriate values have been passed to prepare_graph().
x_axis(pos) Sets the default location of the x-axis. Possible values are 'middle' (default), 'top' or 'bottom'. May not be needed if the appropriate values have been passed to prepare_graph().
y_axis(pos) Sets the default location of the y-axis. Possible values are 'left' (default) and 'middle'. May not be needed if the appropriate values have been passed to prepare_graph().
plot_function(fn) Plots a function of one variable (fn) over the entire range of values for the graph.

Valid colours
'named colour'HTML colour name.
'hex value'Colours are defined using a hexadecimal notation for the combination of Red, Green, and Blue colour values (RGB). The lowest value that can be given to one basic colour is 0 (hex #00). The highest value is 255 (hex #FF or #ff; both upper and lower case letters are accepted). See named colours for examples.
'rgb(red, green, blue)'Colours are defined using a decimal notation for the combination of Red, Green, and Blue colorr values (RGB). The lowest value that can be given to any one of the basic colour (red, green, or blue) is 0. The highest value is 255.
'rgba(red, green, blue, alpha)'Similar to rgb(red, green, or blue), but with the addition of a transparency parameter (alpha). The valid range of values is from 0.0 (fully transparent) to 1.0 (fully opaque). By default, this property is set to 1.0 (fully opaque).
