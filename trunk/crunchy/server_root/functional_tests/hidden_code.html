Hidden code test

This tests to see if code that is hidden by a tutorial writer shows up. The code is hidden as follows: 

>>> <span style="display:none">print "hidden stuff"</span>
>>> print "innocent looking code"

>>> print "hidden stuff"
>>> print "innocent looking code"

>>> print "hidden stuff"
>>> print "innocent looking code"

>>> print "hidden stuff"
>>> print "innocent looking code"
