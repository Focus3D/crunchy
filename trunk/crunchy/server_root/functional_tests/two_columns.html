Two columns test

This document contains a test for two column layout, based on custom css. This is work in progress - it does not behave as desired.

Code only

Python code should be only in one column, aligned left. This is obviously not working.

>>> print 'title = "py_code"'
>>> for i in range(10):
...     print i
...

Interpreter

Interpreter should be on the left, with code sample on the right. This is not working.

>>> print 'title = "interpreter"'
>>> for i in range(10):
...     print i
...

Editor

Editor should be on the left, with code sample on the right. This is working ... if the window is wide enough.

>>> print 'title = "editor"'
>>> for i in range(10):
...     print i
...

Doctest

Editor should be on the left, with code sample on the right. This is working ... if the window is wide enough.

>>> double(3)
6
