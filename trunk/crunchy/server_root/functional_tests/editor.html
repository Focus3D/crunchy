Crunchy Editor Test

Crunchy allows embedding a number of independent editors in a single page. Try running the code by pressing the "execute" button at the bottom.

a = 7*6
print a

You may also want to toggle the editor from a simple textarea into a syntax aware editor. The more complex editor can save and load Python scripts.

A second editor (with code preceded by line numbers) is provided below. Try it out and verify that the variable a is not defined, thus confirming that each editor is running code independently from the others.

for i in range(5):
    print i,
print 
print a

Finally, code from a simulated interpreter session can be extracted and copied into an editor. Try out the following simple example.

>>> x = 7*6
>>> print(x)
42
>>> range(5)
[0, 1, 2, 3, 4]
>>> for y in range(5):
...     print y,
... 
0 1 2 3 4

If you toggle the editor, you may want to try to:

  • save a file; you need to browse to the desired location and click on the file name (or some other file, and then edit the result) to get the path correctly.
  • load a file, again using the browsing utility.
  • save and run a file as an external program.
