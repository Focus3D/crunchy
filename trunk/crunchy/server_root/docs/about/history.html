In The Beginning ...

A Potted Crunchy History

... If I may begin at the beginning. First there is the cherry fondue. This is extremely nasty, but we can't prosecute you for that...
From the Monty Python Crunchy Frog sketch.

In the beginning, there was IT. The original idea for IT had been simmering in André's mind for a few years. The first implementation was inspired by an REPL (read-eval-print-loop) implemented by Robert Brewer using CherryPy. IT was nice and IT was the name suggested by André Roberge's 13 year old daughter Evelyne after he showed her and explained to her that IT was designed to produce Interactive Python Tutorials. However, IT was found to be slightly uninspiring as a name. After asking on the edu-sig list for potential names, IT became Crunchy Frog.

Crunchy Frog (it probably was more of a Tadpole back then) grew slowly until Johannes Woolard joined as a developer, thanks to Google's financial support during the Summer of Code 2006. As the Frog matured and another project was found with the same name, natural selection took its course and, as of version 0.7, this project was renamed "Crunchy".

Since then, much development has occurred, many people have contributed code, and Crunchy has grown, even spanning a new project: Croquant, which means crunchy in French.
