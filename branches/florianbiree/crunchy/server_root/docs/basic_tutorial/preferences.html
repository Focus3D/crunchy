User's preferences

Crunchy has been designed to be highly configurable by the end user. That is you, your name. To see the various configuration options, you could type in help(crunchy) in the interpreter below - or in any interpreter or other Crunchy object that can be used to execute some Python code, decipher the information provided, and figure out which values to choose, relying on Crunchy to tell you when you make a typing mistake.

help(crunchy)

Automatic saving

As soon as you change a value for the preferences, it is saved automatically. However, it is likely that you will have to view a new page, or reload the current one, to see the effects of your changes.
