Crunchy vous souhaite la bienvenue!

Utiliser le menu ci-dessus pour naviguer. "Maison" vous ramènera toujours à cette page. "Quittez Crunchy" fermera la page ou la fenêtre du fureteur en plus de terminer le programme lui-même.

Vous pouvez changer le style de la page à partir du menu de Firefox. Notez que, si vous changer le style, vous devrez le refaire pour chaque page que vous visiterez. À l'avenir, nous prévoyons inclure une option vous permettant de faire un choix global pour le style choisi.

Si vous désirez en savoir plus sur ce que Crunchy peut faire, sélectionner le didacticiel "Utiliser Crunchy" (en anglais seulement).

Si vous lisez cette page sans utiliser Crunchy, les liens du menu ci-dessus ne fonctionneront pas. Vous pouvez utiliser ce lien. À noter que toute la documentation est en anglais pour l'instant.
