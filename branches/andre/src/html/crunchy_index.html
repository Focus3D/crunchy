Welcome to Crunchy!

Use the menu above to navigate. "Home" always brings you back to this page; "Quit Crunchy" closes the browser tab (or window) in addition to ending the program as such.

You can change the page style using Firefox (menu: View --> Page Style. Note that, if you do not want to use the default style, you will have to set your style choice for every page that is viewed. In the future, you will be able to set your preferred option, and Crunchy will remember it.

If you want to learn what Crunchy can do, select the "Using Crunchy" tutorial.

If you are viewing this page without using Crunchy, then the menu above (shown as a list of links) will not be working. You can use this link to go through the "Using Crunchy" tutorial; unfortunately, this will give you a very poor impression of Crunchy's true capabilities.
