Chewy vous souhaite la bienvenue!

Utilisez le menu ci-dessus pour naviguer. "Maison" vous ramènera toujours ici. "Quitter Chewy" terminera le programme Python, vous laissant avec un onglet (ou une fenêtre) à fermer.

Vous pouvez changer le style de la page à partir du menu de Firefox. Notez que, si vous changer le style, vous devrez le refaire pour chaque page que vous visiterez. À l'avenir, nous pré voyons inclure une option vous permettant de faire un choix global pour le style choisi.
