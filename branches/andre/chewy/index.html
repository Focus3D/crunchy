Welcome to Chewy!

Use the menu above to navigate. "Home" always brings you back to this page; "Quit Chewy" ends the Python program, leaving you with a tab (or window) to close.

You can change the page style using Firefox (menu: View --> Page Style. Note that, if you do not want to use the default style, you will have to set your style choice for every page that is viewed. In the future, you will be able to set your preferred option, and Chewy will remember it.
