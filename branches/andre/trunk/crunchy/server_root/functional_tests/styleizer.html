Pygment-based styling Tests

Crunchy uses Pygments to style code. To specify a language, Crunchy uses the same convention that is used by Pygment as given on Pygments available Lexers. For example, a Python code sample can be styled using either: <pre title="PythonLexer"> 

print "Hello World!"

or <pre title="python">

print "Hello World!"
or <pre title="py"> 
print "Hello World!"

As further simple tests, we style some Python code with an embedded <br/> to complicate things.
