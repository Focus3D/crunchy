Pygment-based styling
In this file, we make use of the "show_vlam" option to automatically reveal what options we used to style the code just below it.

If you have throught the user preference page, you may have read that Crunchy uses Pygments to style code. To help you choose a given style, we provide you with a few code samples on this page to see the result of your choice.

If you are a tutorial writer (or if you used the advanced options to override Crunchy's defaults), you might be interested to learn that, to specify a language, Crunchy uses the same convention that is used by Pygment as given on Pygments available Lexers, with a few other "obvious" choices as alternative options.

For example, a Python code sample can be styled using either: 

print "Hello World!"

or 

print "Hello World!"

or 

André = "my name"  # note the syntax error
print "Hello World!"

# Identifiers with non-ascii characters are ok with Python 3!
André = "my name"
print "Hello World!"

You can select the desired style here and reload the page to see its effect.
