What is needed to run Crunchy?

Since you are reading this, chances are that you are already using Crunchy. If that is not the case, in addition to Crunchy you will also need:

  • Python. I do mean Guido van Rossum's Python, not Monty Python nor a real life snake. Chances are it is already installed on your computer; but you do need a version no older than version 2.4. [However, Crunchy is not compatible with Python 3.0.]
  • Firefox. Crunchy may work (at least partially) with some other browsers, but it is currently only tested with Firefox. Also, for security reasons, we recommend that you use a Firefox version no older than 2.0.0.5.

New to Crunchy?

If you are new to Crunchy, we strongly suggest that you first go through the basic tutorial using the links on the left.

You can use the Crunchy Menu at the top right to come back to this page or if you have administrator privileges, to quit Crunchy altogether.
