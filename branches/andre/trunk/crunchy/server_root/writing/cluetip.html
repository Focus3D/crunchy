Inserting Popup Helpers

Inserting popup helpers is fairly straightforward. We make use of the cluetip plugin for jquery.

There are two options available for popup helpers: using information from the title attribute only, or using full web pages. An example of the first one is given by the following html code.

<img 
    title="cluetip This will be the title | with some other | text added." 
    src="/images/help.png">

You need more?

with this: as a live example. Note that the first "|" symbol is used to separate the title from the rest of the text, while others are equivalent to inserting new lines. For this type of popup helper the width is set to 300px.

There is another way to insert popup helpers, using full html pages that are identified by the rel attribute, as in the following code.

<img 
    title="cluetip A title for the web page" 
    rel="/docs/popups/monty1.html" 
    src="/images/help.png"
    style="height:32px;"
>

What does Crunchy uses as a clue?

To identify that a popup helper is requested, Crunchy uses the fact that the first word to appear in the title attribute is cluetip. It removes that word from the title attribute and then proceeds to insert the appropriate html and javascript code to make the whole thing work. Note that Crunchy can use pretty much any html element to insert a popup helper, even this bold text.
