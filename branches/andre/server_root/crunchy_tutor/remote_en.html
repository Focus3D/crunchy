Officially learning Python

To learn about Python, it might be a good idea to through the official Python tutorial on the python.org website.

You are asking Crunchy to do what?

You should always understand any Python code you ask Crunchy to execute. Crunchy is just a tool that saves you the trouble of copying and pasting code you find somewhere before trying it out on your computer. YOU are responsible for asking Python to execute some code - not Crunchy's designers.

Elsewhere on the planet

Perhaps you are a more advanced user, interested in browsing the latest Python blogs. If you do so, you might want to look at them using this:

Safe browsing!

Crunchy is a tool that can be used to try out Python code found in html documents, either locally or on the web. However, we can not assume responsibility for security holes: use at your own risk. If you do find potential security risks, including other tests to use on our security test page, please let us know.

Closer to home
