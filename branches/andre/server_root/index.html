Welcome to Crunchy!

This is a preview version of the new Crunchy. Crunchy has been re-written from the ground up (yet again!). The previous "stable" version is 0.8.2. When this new version is considered "stable", it will become version 0.9 with plans for the 1.0 release soon after. Here are some suggestions as to how you can use Crunchy.

Python interpreter

The first thing you might want to try, is to use the embedded interpreter to write the standard first program. 

>>> print "Hello World!"
"Hello World!"

Officially learning Python

After writing your first Python instructions, you might be interested in going through the the official Python tutorial on the python.org website to learn more about Python.

Elsewhere on the planet

Perhaps you are a more advanced user, interested in browsing the latest Python blogs. If you do so, you might want first to enter crunchy.no_markup = "editor" in the above interpreter prompt before loading up the blogs and possibly trying out some sample Python code you might find there.

http://planet.python.org

If the user has include some Python code inside an html <pre> element, Crunchy will recognize it and make sure you can try it out. Unfortunately, Crunchy sometimes does not do a very good job of processing and rendering non W3C compliant pages for display in the browser. If you find such a page, complain to its creater, not to us! If you do find a W3C compliant page that Crunchy does not interpret properly, then let us know.

Closer to home

What if you already have some Python tutorial (html documents) on your own computer and you would like to use Crunchy to go through them. You can browse and load them here.

Testing, 1, 2, testing, ...

Finally, in the absence of real documentation for Crunchy, you might want to browse the various tests we include. In doing so, if you notice a bug, please let us know.

Enjoy!
