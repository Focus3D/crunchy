Welcome to Crunchy!

This is version 0.9.8.6. We were planning to have version 1.0 released but, since many small improvements are added to Crunchy through various tasks as part of Google's Highly Open Participation contest, we are delaying the release to incorporate these improvements and give a better public recognition to the students that are contributing to Crunchy.

If you are new to Crunchy, you may want to go first through the user tutorial using the links on the left.

You can come back to this page by clicking on the Crunchy Python logo at the top of any page. To quit Crunchy, simply click on the "Quit" link at the top.

New in 0.9.8.6: It is possible to process reStructuredText files the same as plain html files. Check on this page, towards the bottom.

New in 0.9.8.5: External execution using alternate Python version.
