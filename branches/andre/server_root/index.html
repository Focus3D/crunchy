Welcome to the new Crunchy!

Safe browsing!

Crunchy is a tool that can be used to try out Python code found in html documents, either locally or on the web. Before you browse the net with Crunchy, make sure you check this security test page to see if using Crunchy with your browser avoids common security problems.

Even then, we can not assume responsibility for security holes: use at your own risk. If you do find potential security risks, including other tests to use on our test page, please let us know.

If you know what you are doing, you can visit external sites, by typing in the appropriate url. We provide a default one to get you started.

http://docs.python.org/tut/

Remember: before executing any Python code within Crunchy, make sure you understand what it does. Very simple Python programs (even a simple line of code) can be used to delete all the files on your computer.

Changes from 0.9

  • New experimental session logging feature (look on the Tests menu)
  • Bug fix that prevented output containing <...> from being displayed.
  • New way to generate image files that do not require the launch of separate processes.
  • Friendlier tracebacks (for Python beginners).
  • More configuration options.
  • Improved safety.
  • New interpreter options.
  • New logo.
  • More documentation.

Crunchy has been re-written from the ground up (yet again!). This is version 0.9.2, the third release in the 0.9x series; the previous "stable" version was 0.8.2. This new version has many enhancements over the previous one. We are working steadily towards the 1.0 release.

If you are new to Crunchy, you may want to go first through the user tutorial using the links on the left.

You can come back to this page by clicking on the Crunchy Python logo at the top of any page. To quit Crunchy, simply click on the "Quit" link at the top.

Enjoy!
