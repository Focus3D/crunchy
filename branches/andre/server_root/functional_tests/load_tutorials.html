Tutorial loading tests

We provide two type of forms/inputs to load remote or local tutorials. First the remote case, which we take to be the official Python tutorial: http://docs.python.org/tut/tut.html. Of course, one can change the value from the default, to be any valid url.

The second case is for local tutorials. (not implemented yet) Here, due to browser/javascript security reasons, we can not provide a default value.
