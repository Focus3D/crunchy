Crunchy External Execution Tests using different Python version

Suppose you have two different Python versions installed on your system and you would like to try programs using version A while launching Crunchy using version B. You can now do this using the interactive element below. All you have to do is to specify the path (or 'alias') of the Python version you want to use in the input box and then click on a button. For example, on my system, 'python' defaults to the version 2.5 whereas 'python3.0' launches an alpha version of Python 3.0.

The following code will start a Python interpreter in a separate terminal window. 

import code
code.interact()

Here's a second Editor box, which uses identifiers (variable name) with accents; this will only work with Python 3.0+. 

prénom = "André"
print(prénom)
input("Press any key to quit. ")   # raw_input no longer exist in 3.0
