Comprehensive vlam test

In this document, we try to list all possible vlam combinations for reference.

No markup

>>> print 'title = "no_vlam"'

Styling only

>>> print 'title = "py_code"'

>>> print 'title = "py_code linenumber"'

>>> print 'title = "py_code linenumber=99"'

Styling of Python code inside <code> element, such as
print 'title = "py_code"'
print 'title = "py_code linenumber"'
print 'title = "py_code linenumber=99"'
is also possible.

Interpreter

Three examples with standard interpreter

>>> print 'title = "interpreter"'

>>> print 'title = "interpreter linenumber"'

>>> print 'title = "interpreter linenumber=99"'

Similar three examples with isolated interpreter

>>> print 'title = "interpreter isolated"'

>>> print 'title = "interpreter isolated linenumber"'

>>> print 'title = "interpreter isolated linenumber=99"'

Editor

Three examples with input as interpreter sessions (with prompt)

>>> print 'title = "editor"'

>>> print 'title = "editor linenumber"'

>>> print 'title = "editor linenumber=99"'

Similar three examples with input as normal python code (no prompt)

print 'title = "editor"'

print 'title = "editor linenumber"'

print 'title = "editor linenumber=99"'

Similar three examples with no styled Python code

print 'title = "editor no-pre"'

Similar three examples with Python code styled but not copied in the editor

print 'title = "editor no-copy"'

print 'title = "editor linenumber no-copy"'

print 'title = "editor linenumber=99 no-copy"'

Three basic examples

>>> print 'title = "doctest linenumber=99"'
