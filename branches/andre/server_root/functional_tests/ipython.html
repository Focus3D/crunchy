IPython Test

IPython is a fairly advanced replacement for the normal Python console. Unfortunately, trying to use it as an intpreter shell messes up Crunchy rather badly. Assuming you have it installed on your system, to see it in action, execute the following code. 

from IPython.Shell import IPShellEmbed
ipshell = IPShellEmbed(["-colors", "NoColor"])
ipshell()

Bugs

  • It works ... but messes up everything else. Try going to a normal interpreter and entering "42" to see what happens.
