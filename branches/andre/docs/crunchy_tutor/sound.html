Try running the code supplied: you should hear some sounds - assuming you have speakers connected to your computer. Note that you may have to wait a while for processing.

play_song(yesterday)

If you are using Windows, and you want to have more immediate results in terms of sound output, at the cost of using the built-in system beep of lower quality, you can also try the option:

play_song(yesterday, True)

where True is a shortcut for approximate=True. Note that this option is only available when one note (or tone) is played at a given time.

You can use the simple sound module included with Crunchy to demonstrate the concept of beats:

play_tones((440, 441), 4)

as well as the concepts of harmonics (series) closely related to that of musical instruments' timbre. 

play_harmonics(440, 3, (1,)) # pure tone
play_harmonics(440, 3, (1, 0, 1./9, 0, 1./25, 0, 1./49)) # triangular
play_harmonics(440, 3, (1, 0, 1./3, 0, 1./5, 0, 1./7)) # square
play_harmonics(440, 3, (1, 1./2, 1./3, 1./4, 1./5, 1./6, 1./7)) # sawtooth

More details can be found in the Sound Reference.
