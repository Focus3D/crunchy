Welcome to the new Crunchy!

Safe browsing!

Crunchy is a tool that can be used to try out Python code found in html documents, either locally or on the web. However, we can not assume responsibility for security holes: use at your own risk. If you do find potential security risks, please let us know. In the meantime, and before browsing external pages, please click on the first link of FAQ, bugs, etc. to learn about security as implemented by Crunchy. [Unfortunately, Crunchy can not bring you to a specific point on a new page; it always displays the page from the top.]

If you do find potential security risks, please let us know.

Changes from 0.9.1 (last public release)

  • Much improved security.
  • 3 security levels.
  • New interpreter options.
  • New interpreter types.
  • New logo.
  • More documentation.
  • Friendlier tracebacks (for Python beginners).
  • More configuration options.

Crunchy has been re-written from the ground up (yet again!). This is version 0.9.3, the fourth "major" release in the 0.9x series. This new version has many enhancements over the previous one. We are working steadily towards the 1.0 release.

If you are new to Crunchy, you may want to go first through the user tutorial using the links on the left.

You can come back to this page by clicking on the Crunchy Python logo at the top of any page. To quit Crunchy, simply click on the "Quit" link at the top.

Enjoy!
