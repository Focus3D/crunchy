This is a tutorial on the use of Crunchy (version 0.7). Crunchy's goal is to transform plain web-based Python tutorials into amazingly exciting interactive sessions!

What is needed to run Crunchy?
... If I may begin at the beginning. First there is the cherry fondue. This is extremely nasty, but we can't prosecute you for that...
From the Monty Python's Crunchy Frog sketch. Note that the first versions of Crunchy also required to have CherryPy installed; CherryPy is a nice and easy to use app if you ever need it.

Since you are reading this file, chances are that Crunchy is just waiting for you, in the same directory as the one where you've found this file. However, you will also need:

  • Python. I do mean Guido van Rossum's Python, not Monty Python. Chances are it is already installed on your computer.
  • Elementtree. If it is not already installed on your computer, you can find it on effbot's site.

Once these two essential packages are installed, you will be ready to use Crunchy. I asssume you know how to do this (hint: look for crunchy.py). You will know that you have been successful if your browser window takes over and you see this very page appear, but with a much improved appearance. In the mean time, I suggest you read through the next few pages to see how Crunchy transforms a traditional Python tutorial into something that is so much more interesting.
