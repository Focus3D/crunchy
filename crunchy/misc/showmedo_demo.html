This is a demonstration of a presentation made using Crunchy and a pre-alpha release version of crst2s5.

Why is it called crst2s5?

  • S5: Simple Standards-based Slide Show System
S5 was created by Eric Meyer. It consists in a series of style files and some javascript code to transform an html file into a slide show.
  • rst: reStructuredText
  • rst2s5.py: docutils script used to transform a reStructured text document into an S5 html file.
    • based on S5 version 1.1

  • crst2s5.py: similar to rst2s5.py

    • based on a customized version of S5 version 1.2a
    • version 1.2a includes speaker notes & timer

    Show some speaker notes.

    • crst2s5.py includes some custom directives
    • automatic font scaling disabled, etc.

Custom directives

  • New directive:

    .. crunchy:: Very Little Added Markup (or vlam)
This is what enables the embedding of interactive elements with Crunchy

  • New directive:

    .. getpythonsource:: path/to/file/module.object
Uses Python's inspect module to get insert the relevant source code.

getpythonsource tests (1)

The following automatically import the module a_test_file. When the generated html file is viewed with Crunchy the code will be styled and line numbers will be added.:

.. getpythonsource:: a_test_file python linenumber

'''a_test_file.py: a demo file for getsource.py'''

def print_squares(n):
    '''Prints a table of number and their squares'''
    for i in range(n):
        print i, i*i

class Fantastic(object):

    def __init__(self):

    def bye(self):

getpythonsource tests (3)

If the desired object can not be imported, a traceback is produced instead.:

.. getpythonsource:: a_test_file.does_not_exist python

Traceback (most recent call last):
  File "/Users/andre/CrunchySVN/trunk/crunchy/src/plugins/rst_directives.py", line 122, in get_lines
    to_inspect[_obj[i]] = getattr(to_inspect[_obj[i-1]], s)
AttributeError: 'module' object has no attribute 'does_not_exist'

Note that we added "python" at the end to tell Crunchy that we expected a Python source code and to style it (using pygments) appropriately.

Odds and ends

  • This presentation was produced by the following command at a terminal prompt:

    python ../crst2s5.py --theme=ui/showmedo --theme-url=ui/showmedo showmedo_demo.txt showmedo_demo.html

  • Note: if using a dark theme (say, with a black background), select the appropriate pygments theme in Crunchy's preferences.

  • To find out more about rst2s5, please consult http://docutils.sf.net/docs/user/slide-shows.s5.html

  • To get crst2s5 and/or crunchy, go to http://code.google.com/p/crunchy
