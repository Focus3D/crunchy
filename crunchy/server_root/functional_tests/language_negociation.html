Language negotiation

Abstract

Some websites implement a language negotiation system, to provide a content in the visitor's language. This is a part of the content negotiation, and use the Accept-Language header.

Crunchy can forward received Accept-Language headers to a remote site, to get it in the same language than outside Crunchy. You can enable (default) this by:

crunchy.forward_accept_language = True

And disable by:

crunchy.forward_accept_language = False

How to choice the prefered language?

Web browsers allow to set a list of prefered languages, sorted by priority. For Mozilla Firefox, the choice is available in Edit > Preferences > Content > Languages (or Tools > Options > Content Languages, depending of the system).

Sites known to use language negotiation

  • Mozilla Europe
  • The Apache documentation
  • MoinMoin wikis localize some part of the interface (like the login button)
