How does Crunchy perform its magic?

In case you are wondering, we will explain graphically how Crunchy performs it magic. There are four parts to consider.

  • The User starts Crunchy and communication is established.
  • Crunchy sanitizes the html pages requested to remove unwanted elements.
  • Crunchy processes the markup (vlam) to insert appropriate interactive elements.
  • The User requests that some Python code be executed
